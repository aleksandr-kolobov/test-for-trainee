Получить все достопримечательности

после перехода вы получите вывод всех достопримечательностей из базы в формате JSON.

по умолчанию они отсортированы по номеру записи в базе данных.

для сортировки записей по имени можно добавить свойство(параметр запроса) sort=true.

это можно сделать добавив в конец строки запроса "?sort=true" или "?sort=1"

также можно сделать выборку по конкретному городу или по типу достопримечательности

(Здание/Сооружение/Музей/Памятник/Заповедник) нужно указать

варианты type=building, construction, museum, monument, reserve

для этого нужно добавить параметры запроса town= и type= через &

например ?sort=1&town=moskow получим достопримечательности москвы отсортированые по наименованию.

регистр ввода не имеет значения.

для получения отдельной записи по номеру в базе данных, можно добавить номер через слеш

например sights/3 вернет запись с ID=3

Получить все города

после перехода на страницу со списком всех городов,

также можно добавить свойство sort=true, для сортировки по наименованию города

для получения записи отдельного города из базы данных, можно добавить имя или номер через слеш

например towns/3 вернет город с ID=3, или towns/tomsk вернет запись города томск в формате JSON
